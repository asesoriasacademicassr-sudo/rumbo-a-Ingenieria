# Arquitectura 2.0

- Frontend PWA adaptable a celular.
- Autenticación: Supabase Auth.
- Base de datos: PostgreSQL con RLS.
- Archivos: Supabase Storage.
- Tiempo real: Supabase Realtime.
- Atenea: Edge Function; la clave de IA nunca se expone en el navegador.
- Modo demo local para probar la experiencia sin cuentas externas.

## Roles
Administrador controla el sistema. Profesor gestiona cursos, clases, recursos y calificaciones. Alumno accede a cursos, tareas y entregas. Padre ve únicamente a sus hijos vinculados.
